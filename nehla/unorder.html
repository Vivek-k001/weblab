LIST IN HTML

Ordered List

An ordered list in HTML is a list of items numbered sequentially, created using the ol tag.

  1. Decimal Numbers(1,2,3)
  2. Uppercase letters(A,B,C)
  3. Lowercase letters(a,b,c)
  4. Uppercase Roman numerals(I, II, III)
  5. Lowercase Roman numerals(i,ii,iii)

UnorderedList

An unordered list is a list of items where the order does not matter. It is typically displayed using bullet points instead of numbers.

  • Disc
  • Circle
  • Square
  • None

Discription List

A description list (also known as a definition list) is a list used to group terms with their corresponding descriptions or definitions. It's useful for glossaries, dictionaries, or any content where you want to pair terms with explanations.

Eg:-
HTML-
A markup language used to structure web pages.
CSS-
A style sheet language used to describe the presentation of web pages.
